# Drivers (собственные драйверы)

- Назначение: реализация собственных драйверов поверх `ITransport` и наших абстракций.
- Рекомендуемая структура: `src/Drivers/<Vendor>/<DeviceModel>/`.

Чеклист разработки драйвера:
- Собрать спецификацию/сниффинг протокола и форматов.
- Спроектировать кодек/фреймер (encode/decode), таймауты, retry.
- Реализовать обмен через `ITransport` (TCP/Serial/USB) и адаптер `IDevicePlugin`.
- Вернуть унифицированный контракт устройства (`ICamera`/`IPrinter`/`IGate`).
- Добавить контрактные и интеграционные тесты.
